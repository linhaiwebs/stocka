# JP Stock AI Template - 日本通版

## 概要
日本無料株教育向けのランディングページテンプレート。Google Ads日本市場向けに最適化されており、投資助言ではなく教育目的に焦点を当てています。

## テンプレート情報
- **ID**: jp-stock-ai
- **名前**: 日本通版
- **カテゴリー**: 日本無料株
- **バージョン**: 1.0.0

## 必要な画像ファイル

このテンプレートを使用するには、以下の画像ファイルが必要です:

### top1.png
- **場所**: `/public/top1.png`
- **説明**: AI選定株の実績データを示す画像
- **内容**: 市場全体の騰落率とAI選定優良株の実績比較
- **推奨サイズ**: 1200x800px 程度

画像をプロジェクトに追加する手順:
```bash
# 画像を public ディレクトリにコピー
cp /path/to/top1.png /tmp/cc-agent/63358226/project/public/top1.png
```

## 機能

### 1. トップバナー (黒背景)
- AI選定の最新株式情報を強調
- 呼吸アニメーション付き赤色インジケーター

### 2. 基本情報セクション (グレー背景)
- 優良銘柄リストの紹介
- LINE連携CTA ボタン
- 長期保有向けの投資教育コンテンツ

### 3. 背景紹介セクション (ブルー背景)
- AI技術の優位性を3つの理由で説明
- 情報の鮮度、感情制御、投資判断サポート

### 4. サービスカードセクション (深ブルー背景)
- AI解析システムの4つの主要機能
- データ解析、市場心理、資金追跡、暴落予測

### 5. 信頼性セクション (シルバーグレー背景)
- 分析精度の可視化
- 実績データの表示 (top1.png)

### 6. FAQ セクション
- よくある質問への回答
- 透明性の高い情報提供

### 7. 固定CTA (ページ下部)
- 常時表示の転換ボタン
- LINE連携による無料レポート提供

## コンバージョントラッキング

このテンプレートは2つの主要なCTAボタンを実装:

1. **First CTA** - セクション2の「高配当·優良銘柄リストを受け取る」
2. **Main CTA** - 固定フッターの「無料の優良株レポートを受け取る」

両方のボタンは以下のトラッキングを実行:
- クリックイベントの記録
- コンバージョンの記録
- トラフィックリンクURLへのリダイレクト（URLが存在する場合のみ）

## Google Ads コンプライアンス

このテンプレートは日本市場のGoogle Ads ポリシーに準拠するよう設計:
- 投資助言ではなく教育目的を明確化
- 過去の実績に関する適切な免責事項
- 誠実で透明性の高い情報提供
- 誇大広告の回避

## 使用方法

### テンプレートの登録
```bash
npm run generate:templates
```

### ビルド
```bash
npm run build
```

### 開発環境での実行
```bash
npm run dev
```

テンプレートは `/l/[traffic-link-slug]` または `/template/[template-id]` でアクセス可能です。

## カスタマイズ

テンプレートの内容は `index.tsx` 内の以下の定数を編集することでカスタマイズ可能:

- `DIRECTORY_ITEMS` - 3つの理由セクションの内容
- `SERVICE_CARDS` - AI解析システムの機能カード
- `FAQ_ITEMS` - よくある質問と回答

## 技術仕様

- **フレームワーク**: React + TypeScript
- **スタイリング**: Tailwind CSS
- **アイコン**: Lucide React
- **アニメーション**: CSS Keyframes
- **トラッキング**: Template Context API

## 注意事項

- トラフィックリンクが設定されていない場合、CTAボタンはリダイレクトを実行しません
- すべてのコンバージョンとクリックは正しくトラッキングされます
- 画像ファイル (top1.png) は必ず public ディレクトリに配置してください
